# ロト7 予知の天文台

ロト7を「多角的な分析」と「日々の気持ち」で楽しむ、家族向けの予測・記録アプリ（Streamlit製）。
当選確率は変わりません。**娯楽として、運命を楽しみながら正直に積み上げる相棒**です。

> このリポジトリは公開ですが、**秘密情報（APIキー・スプレッドシートURL・個人情報）は含まれません**。
> それらはすべて Streamlit / GitHub の Secrets と、非公開の Google スプレッドシートの中にあります。

---

## 毎週の使い方（手動はこの2つだけ）

- 🌍 **予測の積み上げ**：その日の気持ちを書いて積み上げる（探索モード＝30口を別々の可能性に分散）
- 🎯 **金曜の最終決断**：口数を選んでレポートを出し、購入する

## 自動でやること（GitHub Actions・操作不要）

- **金 05:00 JST**：他サイト予想を収集（節約モード）
- **土 09:00 JST**：結果取得＋全期間更新（天気つき）＋自動採点＋総監督レポート生成（→「反省ログ」へ保存）

## 主な機能

- 週次サイクル・ダッシュボード（今日のミッション表示）
- 全方位レンズ（過去頻度・引っ張り・スライド・環境共鳴・量子シード・AI直感・他サイト成績・人気回避・縁起日 ほか）
- 当たっている予想サイトの成績ランキング＆精度反映
- 通算成績／バックテスト（忖度なしの有効性検証）／天気×結果の検証
- 万能AI占い師（プライベート相談室つき）

## セットアップ（値は各自で。ここには載せません）

### 1) Streamlit Secrets（アプリ用）
- `ANTHROPIC_API_KEY` … Claude APIキー
- `GCP_SERVICE_ACCOUNT_JSON` … Googleサービスアカウントの認証JSON
- `SPREADSHEET_URL` … データ保存先スプレッドシートのURL
- （任意）`ANTHROPIC_MODEL` … 既定 `claude-sonnet-4-6`（最安は `claude-haiku-4-5`）
- （任意）`FORTUNE_PRIVATE_PASSCODE` … 占いのプライベート相談室の合言葉

### 2) GitHub Secrets（自動運用用 / Settings → Secrets and variables → Actions）
- `ANTHROPIC_API_KEY` / `GCP_SERVICE_ACCOUNT_JSON` / `SPREADSHEET_URL`（Streamlitと同じ値）

### 3) スプレッドシート側
- サービスアカウントのメールに編集権限を共有
- 「予想サイトURL」シート（A1=`URL`, A2〜に予想サイトURL）※アプリのボタンで自動作成も可

## 運用メモ

- **キーは Streamlit と GitHub の2か所**。作り直したら両方を更新。
- **Anthropicのクレジット**を切らさない（管理画面で上限設定推奨）。0でも結果取得・採点・各種分析（無料）は継続。
- 他サイト収集は「節約モード（安）」が自動。「発掘モード（ウェブ検索）」はコスト高め＝新サイト探しの時だけ。
- GitHub Actions は**約60日リポジトリ無操作で自動停止**（メールから再有効化）。

## 構成

- `app.py` … Streamlit本体
- `auto_update.py` … 自動運用スクリプト（headless）
- `.github/workflows/auto-update.yml` … 定期実行スケジュール
- `requirements.txt` … 依存パッケージ
